💬 Simple Dialogue System for Unity
這是一個專為 Unity 設計的輕量化對話系統。

👤 關於作者 (About Author)
此專案由 Li Zhihua (Lee-Mao) 開發。我是一名擁有 Unity 實戰經驗的遊戲開發工程師，曾擔任獨立遊戲 《卜算子 (Bu Suan Zi)》 的核心開發主程式。該作品目前已於 Steam 平台正式發布。

📖 專案簡介 (Project Overview)
本對話系統提取了我在開發作品時處理大量文本與劇情的實務經驗。設計目標是建立一個低耦合、易擴充的架構，讓開發者能專注於劇情創作。

🚀 核心功能 (Features)
實戰級穩定性：基於已發布作品的開發邏輯優化，確保對話切換與數據讀取的流暢性。

分支選擇系統：支援多樣化的對話選項，並能根據選擇觸發不同的劇情走向。

進階文字表現：

內建打字機效果 (Typewriter Effect)。

支援角色表情與頭像自動關聯切換。

🛠 技術細節 (Technical Details)
Unity 版本：2022.3 LTS 或更新版本。

📥 如何使用 (How to use)
獲取專案：

Bash
git clone https://github.com/Lee-Mao/Simple-Dialogue-System.git
安裝環境：請確保已安裝 Git LFS 以正確下載大檔案。

開啟專案：在 Unity 中開啟專案，並查看 Assets/Scenes/dialogUI.unity 了解系統配置與觸發流程。
